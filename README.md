# 📚 Miniguia de Estudos com NotebookLM: Henry Ford e o Ford Model T

## 📖 Sobre o projeto

Este projeto foi desenvolvido como parte do desafio da DIO utilizando o **NotebookLM** como ferramenta de aprendizagem baseada em Inteligência Artificial.

O tema escolhido foi **Henry Ford**, o **Ford Model T** e a evolução dos sistemas de produção industrial. O objetivo foi compreender como a inovação tecnológica e os modelos de gestão influenciaram a indústria automobilística e a administração moderna.

---

* Estudar a história de Henry Ford.
* Entender a criação do Ford Model T.
* Compreender o conceito de Fordismo.
* Comparar o Fordismo com o Lean Manufacturing.
* Utilizar o NotebookLM para organizar conhecimentos a partir de diferentes fontes.

Esses ajustes permitiram compreender melhor como construir prompts mais eficientes.

# 📖 Miniguia de Estudos

## Henry Ford

Henry Ford revolucionou a indústria automobilística ao tornar o automóvel acessível para a população por meio da produção em massa.

---

## Ford Model T

O Model T foi projetado para ser simples, resistente, barato e de fácil manutenção.

Sua produção em larga escala reduziu drasticamente o preço dos automóveis e popularizou seu uso.

---

## Linha de Montagem

A introdução da linha de montagem móvel reduziu o tempo de fabricação de um veículo de aproximadamente **12,5 horas para apenas 93 minutos**, aumentando significativamente a produtividade.

---

## Fordismo

Características:

* Produção em massa
* Produtos padronizados
* Alta especialização do trabalho
* Hierarquia organizacional
* Salários elevados para estimular o consumo

---

## Lean Manufacturing

O modelo Lean surgiu posteriormente buscando maior flexibilidade na produção.

Principais características:

* Produção conforme a demanda
* Redução de desperdícios
* Controle de qualidade durante todo o processo
* Participação ativa dos trabalhadores
* Utilização do sistema Kanban

---

# 📚 Glossário

| Termo               | Definição                                                                      |
| ------------------- | ------------------------------------------------------------------------------ |
| Fordismo            | Sistema baseado na produção em massa de produtos padronizados.                 |
| Lean Manufacturing  | Sistema de produção focado na eliminação de desperdícios.                      |
| Kanban              | Método visual para controlar o fluxo de produção.                              |
| Just-in-Time        | Produção realizada somente quando necessária.                                  |
| Economias de Escala | Redução do custo por unidade com aumento da produção.                          |
| Linha de Montagem   | Processo produtivo dividido em etapas sequenciais.                             |
| Vanadium Steel      | Liga metálica utilizada por Henry Ford para aumentar a resistência do Model T. |
